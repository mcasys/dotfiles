# Customized for mc@sys needs

# Installation
### Download and change folder
```
cd ~
git clone https://github.com/mcasys/dotfiles.git

cd dotfiles
```
### Run install
```
./install.sh
```

# Uninstallation

### Change to dotfiles folder
```
cd ~/dotfiles
```

### Run uninstaller
```
./uninstall.sh
```
